Designed to bypass standard software vulnerabilities and operate completely out-of-process.

PCIe Video Capture

Loads frames directly from PCIe capture cards. Bypasses standard OS screen-grabbing APIs, dropping capture latency to under 5ms.

Makcu Hardware Emulation

Communicates directly with the Makcu serial microcontroller. Mouse signals are translated into native hardware inputs, so no software drivers needed.

Automated Target Triggering

Fires input actions automatically when a designated target is detected. Includes trigger delay and custom spatial offsets for precision mapping.

Recoil Control

Compensates dynamically for vertical and horizontal feedback drift. Applies preset correction curves to offset physical recoil patterns, ensuring stable coordinate targeting during continuous input cycles.

Dual Profile Mapping

Supports two completely independent hotkey profiles. Allows you to map mouse speeds, target weights, and height percentages to different analysis scenarios dynamically.

Hands-Free Snapshot Training

Records video feed screenshots with automatic YOLO labeling. Simply enable Snapshot mode, and the AI will compile your training dataset automatically.
